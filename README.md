🎵 Llano Music

La música del llano venezolano

Llano Music es una aplicación Android dedicada a preservar, difundir y promover la música llanera venezolana.

El proyecto forma parte del ecosistema digital de Producciones Lcarballog, integrando artistas, canciones y contenido multimedia en una sola plataforma.

Características

- 🎵 Reproductor de música llanera.
- 🎤 Catálogo de artistas.
- 🖼️ Fotografías de artistas.
- ❤️ Lista de favoritos.
- 🔍 Búsqueda por artista o canción.
- 📻 Futuras radios llaneras.
- 🎥 Integración con YouTube.
- 📱 Diseño moderno para Android.
